Project Details

we build a classifier that detects whether the news is fake or not. This is a binary classification problem. We preprocess the text data from our dataset using TF-IDF Vectorizer.

Description

  • Relevant Feature selection using pandas and Vectorizing them using Tfidvectorizer.

  • Use Passive Aggressive Classifier algorithm and got 93.61% accuracy model.

  • Save the model and implement in the Streamlit app.
